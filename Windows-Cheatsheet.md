## Snapping

### Windows 10
- Move window around <kbd>Win</kbd> + <kbd>arrow keys</kbd>
- Dynemic rezise <kbd>Win</kbd> + <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>arrow keys</kbd>
- Default Windows snap <kbd>Win</kbd> + <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>arrow keys</kbd>
